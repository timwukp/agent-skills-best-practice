# Lab Exercises: Additional Practice

## Bonus Exercise A: Refactor a Skill

Take the Lab 1 skill and refactor it to:
1. Add references/ directory with stack-specific templates
2. Add a validation script in scripts/
3. Add negative triggers to the description

## Bonus Exercise B: Multi-Skill Interaction

Design two skills that work together:
- Skill 1: Generates code
- Skill 2: Reviews generated code

How do you ensure they don't conflict?

## Bonus Exercise C: Enterprise Hardening

Take any lab skill and add:
- Audit trail logging
- Input validation
- Rollback procedures
- Compliance checks
