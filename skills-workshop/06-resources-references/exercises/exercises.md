## Exercise 6.4: Peer Review

Exchange skills with a partner. Review using this checklist:

- [ ] Valid frontmatter (name, description)
- [ ] Description includes WHAT and WHEN
- [ ] Instructions are clear and actionable
- [ ] Error handling present
- [ ] Triggers correctly on test prompts
- [ ] Doesn't over-trigger

Provide 3 specific improvement suggestions.
