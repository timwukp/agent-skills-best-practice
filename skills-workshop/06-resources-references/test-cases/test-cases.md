# Chapter 6 Test Cases

### Test 1: Create a skill from scratch
Given a use case description, create a complete skill in 15 minutes.

### Test 2: Debug a broken skill
Given a skill with 3 intentional errors, find and fix all of them.

### Test 3: Improve a description
Given a weak description, rewrite it to trigger correctly.

### Test 4: Deploy and verify
