# Chapter 5 Templates

## Troubleshooting Checklist

### Skill Won't Trigger
- [ ] Description includes specific trigger phrases
- [ ] Description says WHAT and WHEN
- [ ] Not too generic
- [ ] SKILL.md spelling is exact (case-sensitive)
- [ ] Folder is in correct location

### Skill Over-Triggers
- [ ] Add negative triggers ("Do NOT use for...")
- [ ] Make description more specific
- [ ] Clarify scope boundaries

### MCP Connection Fails
- [ ] MCP server is connected and running
- [ ] Authentication valid (API keys, OAuth)
- [ ] Tool names match exactly (case-sensitive)
- [ ] Test MCP independently first

### Instructions Not Followed
- [ ] Instructions are concise (not verbose)
- [ ] Critical items at top
- [ ] Using bullet points / numbered lists
- [ ] No ambiguous language

## Pattern Selection Guide

| Need | Pattern |
|------|---------|
| Multi-step in order | Sequential Workflow |
| Multiple services | Multi-MCP Coordination |
| Quality improvement | Iterative Refinement |
| Conditional logic | Context-Aware Selection |
| Specialized knowledge | Domain-Specific Intelligence |
