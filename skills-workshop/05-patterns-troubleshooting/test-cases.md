# Chapter 5 Test Cases

## Test Case 5.1: Sequential Workflow Completes
Run a multi-step skill end-to-end. Verify each step executes in order and data passes between steps.

## Test Case 5.2: Error Handling Works
Simulate a failure at Step 2 of a 4-step workflow. Verify rollback occurs.

## Test Case 5.3: MCP Independent Test
Call MCP tool directly without skill. Verify it works before debugging skill.

## Test Case 5.4: Iterative Refinement Converges
Run iterative skill. Verify quality improves each iteration and loop terminates.

## Test Case 5.5: Context-Aware Selection
Provide different inputs. Verify skill selects correct variant/reference each time.
